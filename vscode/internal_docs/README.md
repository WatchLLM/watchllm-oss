# Internal documentation (watchllm/vscode)

Agent-facing and planning context for the WatchLLM VS Code extension repo. These files are **not** public product docs.

## Layout

| File | Purpose |
|------|---------|
| `IMPORTANT.md` | VS Code extension spec: scope, architecture rules, task gates |
| `PROMPT_TEMPLATE.md` | Copy-paste daily task prompts for agent sessions |
| `README.md` | This index |

## Public vs internal

- **`internal_docs/`** — planning and agent workflow (this directory)
- **`../docs` repo** — ecosystem-wide architecture, RFCs, and shared specs
- **Repo root** — `README.md` (public), `tasks.md` (active gate tracker)
